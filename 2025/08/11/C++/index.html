1. C++初识

1.1 注释

  1. 单行注释：//或CTRL+/（快捷键）

  2. 多行注释：/内容/

1.2 变量

语法：数据类型 变量名 = 初始值

int a = 10;

1.3 常量

  1. **#define#**宏常量： #define 常量名 常量值
  • ==通常在文件上方定义==，表示一个常量

#define day 7

  1. const修饰的变量： const 数据类型 常量名 = 常量值
  • ==通常在变量定义前加关键词const==，修饰该变量为常量，不可修改

const int month = 12;

1.5 标识符命名规则

  • 不是关键字

  • 只能是字母、数字、下划线组成

  • 第一个字符为字符或下划线

  • 字母区分大小写

2. 数据类型

2.1 整型

  • 区别在于所占内存空间不同
数据类型 占用空间 取值范围
short（短整型） 2字节 -2^15~2^15-1
==int（整型）== 4字节 -2^31~2^31-1
long（长整型） Windows为4字节，Linux为4字节（32位），8字节（64位） -2^31~2^31-1
long long（长长整型） 8字节 -2^63~2^63-1

2.2 sizeof关键字

  • 统计数据类型所占内存大小

sizeof(数据类型/变量)

2.3 实型/浮点型

  • 用于表示小数（默认情况下输出小数，会显示6位有效数字）
  1. 单精度float

float f1 =3.14f;

cout << “f1=” << f1 << endl;

  1. 双精度double

double d1 =3.14;

cout << “d1=” << d1 << endl;

数据类型 占用空间 有效数字范围
float 4字节 7位
double 8字节 15~16位

3.科学计数法

float f2 = 3e2;//3*10^2

2.4 字符型

char ch =‘a’;

  • 用单引号将字符型变量括起来，且引号内只能有一个字符

  • c和cpp中字符型变量只占1个字节

  • 字符型变量是将对应的ASCII编码放入存储单元(a-97 A-65)

cout << (int) ch << endl;//输出字符的ASCII编码

2.5 转义字符

  • 用于表示不能显示出来的ASCII字符

  • 常用：\n(换行符) \（反斜杠） \t（水平制表）

cout <<“hello world\n”;

cout <<“\”<< endl;

cout <<“aaa\thelloworld”<< endl;

2.6 字符串型

1.C风格字符串：char 变量名[] = “字符串值”

C风格字符串要用双引号括起来

2.C++风格字符串：string 变量名 = “字符串值”

注意包含一个头文件 #include < string >

2.7 布尔类型 bool

bool类型只有两个值,占一个字节

  • true - 真（本质是1）

  • false - 假（本质是0）

2.8 数据的输入

cin>>变量

3. 运算符

3.1 算数运算符

  • 用于处理四则运算
运算符 术语 示例 结果
+ 正号 +3 3
- 负号 -3 -3
+ 加 10+5 15
- 减 10-5 5
* 乘 10*5 50
/ 除 10/5 2
% 取模（取余） 10%3 1
++ 前置递增 a=2;b=++a; a=3;b=3;
++ 后置递减 a=2;b=a++; a=3;b=2;
– 前置递减 a=2;b=–a; a=1;b=1;
– 前置递减 a=2;b=a–; a=1;b=2;

  • 两个整数相除会把小数部分去除掉

  • 小数不能进行取模运算

前置递增：

int a = 10;

++a

后置递增：

int b = 10;

b++

  • 前置先加1后运算，后置先运算后加1

3.2 赋值运算符

运算符 术语 示例 结果
= 赋值 a=2;b=3; a=2;b=3;
+= 加等于 a=0;a+=2; a=2;
-= 减等于 a=5;a-=3; a=2;
*= 乘等于 a=2;a*=2; a=4;
/= 除等于 a=4;a/=2; a=2;
%= 模等于 a=3;a%=2; a=1;

3.3 比较运算符

运算符 术语 示例 结果
== 相等于 4==3 0
!= 不等于 4!=3 1
< 小于 4<3 0
> 大于 4>3 1
<= 小于等于 4<=3 0
>= 大于等于 4>= 1

cout << (a==b) << endl;

3.4 逻辑运算符

运算符 术语 示例 结果
! 非 !a 如果a假则!a真；反之
&& 与 a&&b 如果a和b都为真则结果为真，否则为假
|| 或 a||b 如果a和b有一个为真则结果为真，二者都为假结果为假

在C++中除了0都是真,真变假，假变真

4. 程序流程结构

  • 三种：顺序结构、选择结构、循环结构

4.1 选择结构

4.1.1 if语句

  1. 单行格式if语句：if（条件）{条件满足执行代码}
  1. 多行格式if语句：if（条件）{条件满足执行的语句}else{条件不满足执行的语句}；
  1. 多条件的if语句：if（条件1）{条件1满足执行的语句}else if（条件2）{条件2满足执行的语句}……else{都不满足执行的语句}
  1. 嵌套if语句
  • 练习案例：三只小猪称体重

有三只小猪abc，输入各自体重并判断谁最重

4.1.2 三目运算符

语法：表达式1 ? 表达式2 : 表达式3

解释：

表达式1为真，执行表达式2，并返回结果

表达式1为假，执行表达式3，并返回结果

4.1.3 switch语句

作用：执行多条件分支语句

  • if和switch的区别？

switch的缺点：判断时候只能是整型或字符型，不可以是区间
switch的优点：结构清晰，执行效率高

4.2 循环结构

4.2.1 while循环

语法：while（循环条件）{循环语句}

解释：只要循环条件的结果为真，就执行循环语句

while循环练习案例：猜数字

系统随机生成一个1到100之间的数字，玩家进行猜测，如果猜错，提示玩家过大或过小，如果猜对恭喜胜利，并退出游戏

4.2.2 do…while循环语句

语法：do{循环语句}while(循环条件);

注意：与while的区别在于do…while会先执行一次循环语句，再判断循环条件

练习案例：水仙花数

水仙花数是指一个3位数，它的每个位上的数的三次幂之和等于它本身，利用do…while求出所有三位数中的水仙花数

4.2.3 for循环语句

语法：for(起始表达式;条件表达式;末尾循环体){循环语句;}

练习案例：敲桌子

从1开始到100，如果数字个位含有7，或者数字十位含有7，或者该数字是7的倍数，打印敲桌子，其余数字直接打印输出
